# Inventory-Management-System
It is an basic inventory management system using JSON module in python.
Inventory Mgt system for 60 product Id's with 6 attributes each using json module.
For updating the existing record, first the product ID entered by the user is checked. 
If the product ID is valid, then the updation can be done to each attributes namely product name, brand, category,weight, price, and quantity. 
If the product is invalid, then it message with "Product I'd is wrong OR no such product is available with this product I'd".
For the addition of new products to the record, then the user required to provides all the possible attributes.
For sells, the entered product ID is checked with product ID in record data. 
Then, the amount of stock present is checked with amount of stock in demand. Subsequently, the change in product quantity can be reflected in stock.     
For invoice, the user required to enter customer name and phone number and provides the details of purchased with amount inclusive of taxes.
